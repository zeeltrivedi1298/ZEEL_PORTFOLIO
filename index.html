This Portfolio
This websites dynamic background was created using the Threejs library. The center planet, its wireframe halo and the tetrahedron 'asteroids' are all rendered seperately and animated with random offsets and rotations that are different each time the site is loaded. The layout is a single column design of flexboxes that contain reactive content cards.

Relational Database
This project was done for the databases course at CSUF. I designed and developed a relational database using mySQL to store student and professor personal data, classes, and grades. Additionally, I built a simple PHP front-end to facilitate queries and display their results. This projects front-end and databse were hosted on CUSF's linux server environment for the duration of the class.

Github Repository
React Recipe App
This website is a recipe and cooking web app that uses the Spoonacular API to search for and display recipes. It was designed to have a responsive front-end with the React.js library. The same recipe card structure is used when searching for recipes, and viewing saved recipes. Saved recipes are stored locally as cookies to persist between sessions.
